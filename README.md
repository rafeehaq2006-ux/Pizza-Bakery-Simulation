# Pizza-Bakery-Simulation
This project simulates a pizza bakery. The bakery gets resources and then converts these to other resources and then finally by combining the resources we get pizzas which we can sell and earn credits. With the credits we can buy more producers (these give us resources), or more converters (these convert resources into a different resource) or more customers (per tick/cycle of the simulation we can sell to more customers at a time if we have more customers and therefore earn more credtis per tick/cycle). 

We are able to produce a text-based graph that shows the amount of a resource of your choice against time. You are also able to save and load the current proggress of the simulation, using text files and storing the information in a csv format.
